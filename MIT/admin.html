About Demo Mode

      

        This LagerSync instance runs entirely in your browser using localStorage. 
        No server or database required—perfect for GitHub Pages or local testing.
      

      
    
        
  • Data storage: Browser localStorage (cleared when cache is cleared)
    • 
        
  • Passwords: Plain-text only (never do this in production!)
    • 
        
  • Real-time: No syncing across devices or browsers
    • 
        
  • Notifications: No email or SMS in demo mode
    • 
      

      

        ⚠️ This is a prototype demo only. For production, use a real backend with MySQL, proper authentication, and security measures.
